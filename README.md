# 📄 AI Resume Analyzer

🚀 **AI Resume Analyzer** is a smart web application that analyzes
resumes and provides intelligent feedback to improve job applications.\
It extracts skills, evaluates resume structure, and suggests
improvements to help candidates create **ATS‑friendly resumes** and
increase their chances of getting shortlisted.

🌐 **Live Demo:**\
https://ai-resume-analyzer-dun-gamma.vercel.app/

------------------------------------------------------------------------

# ✨ Features

🔍 **Resume Parsing** - Extracts information from uploaded resumes
(PDF).

📊 **Skill Detection** - Identifies key skills present in the resume.

🎯 **Improvement Suggestions** - Provides tips to improve resume
quality.

📈 **Job Role Recommendation** - Suggests potential career roles based
on detected skills.

🧠 **AI Powered Insights** - Analyzes resume content and highlights
missing elements.

📑 **ATS Optimization Tips** - Suggests improvements for Applicant
Tracking Systems.

------------------------------------------------------------------------

# 🖥️ Demo

Upload your resume and get instant analysis.

**Steps:** 1. Upload your resume (PDF) 2. The system analyzes the

# 🚀 Installation

Clone the repository

``` bash
git clone https://github.com/yourusername/ai-resume-analyzer.git
cd ai-resume-analyzer
```

Install dependencies

``` bash
pip install -r requirements.txt
```

Run backend

``` bash
python main.py
```

Run frontend

``` bash
npm install
npm run dev
```
